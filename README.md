# 📊 База Данных учёта договоров IT-аутсорсинговой компании

Профессиональный веб-проект на Flask и SQLite, разработанный в рамках учебной практики. Система автоматизирует учёт клиентов, ИТ-услуг, договоров, задействованных сотрудников и финансовых транзакций (платежей) для аутсорсинговой компании.

---

## 🛠 Стек технологий

*   **Backend:** Python 3.x, Flask
*   **Database:** SQLite 3 (с поддержкой Foreign Keys и каскадного удаления)
*   **Frontend:** HTML5, Jinja2 шаблонизатор, Bootstrap 5 (Responsive UI)

---

```text

```

## 🗄 Структура Базы Данных (6 связанных таблиц)

База данных построена по принципам реляционной модели с соблюдением ограничений целостности (`NOT NULL`, `CHECK`) и каскадным удалением зависимых строк (`ON DELETE CASCADE`).

1.  **`clients` (Клиенты)** — учёт ИТ-компаний и контрагентов (название, контакты, e-mail, телефон).
2.  **`services` (Услуги)** — каталог предоставляемых ИТ-услуг с контролем стоимости часа (`CHECK(price_per_hour > 0)`).
3.  **`employees` (Сотрудники)** — реестр ИТ-специалистов компании с указанием оклада.
4.  **`contracts` (Договоры)** — ключевая таблица, связывающая клиентов и услуги с фиксацией выделенных часов и статуса договора.
5.  **`contract_details` (Детали договоров)** — таблица связи «многие-ко-многим», закрепляющая сотрудников за конкретными проектами и определяющая их роль.
6.  **`payments` (Платежи)** — финансовый учёт транзакций по договорам с контролем суммы платежа.

## 🚀 Инструкция по запуску

Проект запускается в одну команду и не требует сложной предварительной настройки СУБД.

### 1. Клонирование репозитория и переход в папку
```bash
git clone https://github.com
cd db-outsource-practice
```

### 2. Установка зависимостей
Установите микрофреймворк Flask (драйвер SQLite встроен в Python по умолчанию):
```bash
pip install flask
```

### 3. Запуск веб-приложения
Запустите сервер разработки:
```bash
python app.py
```

После запуска база данных `database.db` создастся **автоматически**, применит структуру из `init_db.sql` и наполнится демонстрационными данными.

### 4. Открытие в браузере
Перейдите по адресу:
👉 [http://127.0.0](http://127.0.0)

## 📈 Реализованный функционал в веб-интерфейсе

*   **Комплексный отчет (INNER JOIN):** Вывод сводной таблицы договоров на главной странице с автоматической подгрузкой названий компаний-клиентов из связанной таблицы.
*   **Финансовая аналитика (SUM):** Расчет общей выручки компании в реальном времени на основе агрегирующей функции по фактически оплаченным транзакциям.
*   **Интерактивный INSERT:** Полноценная валидируемая HTML-форма на Bootstrap 5 для добавления новых записей в базу данных без прямого доступа к терминалу.
